# Firefly RPC

Firefly RPC is a project that enables Discord RPC integration.

## Step-by-Step Installation

1. **Install Node.js**  
   Download and install Node.js from the [official website](https://nodejs.org).

2. **Install npm**  
   npm is usually bundled with Node.js. If not, you can install it separately from the [npm documentation](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm).

3. **Install libraries and start the project**  
   - Install libraries:  
     ```bash
     npm install 
     ```
   - Start the project:  
     ```bash
     node test.js
     ```
   - Or you can run these command to complie to an exe file:
      ```bash
      npm install -g pkg
      pkg test.js --targets node18-win-x64 --output firefly.exe
      ```
